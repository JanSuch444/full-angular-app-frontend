- Status Page – Frontend (Angular)

Tato aplikace slouží jako klientská část pro zobrazování stavu služeb (status panel s proužky). Data se asynchronně načítají z PHP API.

-- Funkcionalita
- Zobrazení historie stavů (posledních 90 záznamů) ve formě barevných čtverečků.
- Tooltip s přesným časem zápisu a textem stavu po najetí myší na proužek.
- Automatická aktualizace dat.

-- Jak spustit lokálně
1. Naklonuj repozitář: `git clone <odkaz>`
2. Nainstaluj závislosti: `npm install`
3. Spusť vývojový server: `ng serve`
4. Aplikace poběží na `http://localhost:4200/`

-- Technologie
- Angular
- SCSS / TypeScrip
